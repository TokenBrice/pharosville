# Claude Guide

Canonical instructions for this repository live in `AGENTS.md`.
Use this file only as a compatibility pointer.

## Fast Start

1. Read `AGENTS.md`.
2. Read `docs/pharosville/AGENT_ONBOARDING.md`.
3. Run `npm run onboard:agent`.

## Scratch And Artifacts

- Use `/agents/` for planning artifacts.
- Use `/outputs/` for temporary screenshots, renders, and test sprites.
- Legacy `/output/` content may exist, but new scratch artifacts should go under `/outputs/`.
